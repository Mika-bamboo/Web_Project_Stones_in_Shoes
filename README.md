# 🪨 Stone-In-Shoe Simulator

**Why do stones always end up inside your shoes?**

An interactive physics simulation web app that visualizes how small stones enter different types of footwear during walking.

> 🚧 Under development — stay tuned

## What Is This?

Everyone has experienced it: no matter what shoes you wear, walking on a gravel path inevitably results in a stone inside your shoe. The only exception? Tall boots.

This project uses physics simulation to explain the phenomenon, covering three core mechanisms:

- **Kick-Up Effect** — the outsole launches stones upward during toe-off
- **Bellows Effect** — the collar gap opens and closes with each step, creating micro-suction
- **Vibration Migration** — heel-strike shockwaves inch stones on the collar edge inward

## Features

- 🚶 2D gait cycle animation with real-time physics
- 👟 Multiple shoe types (sandal / sneaker / oxford / tall boot)
- 🎛️ Adjustable parameters (walking speed, stone size, surface conditions)
- 📊 Stone entry rate statistics and cross-type comparison charts
